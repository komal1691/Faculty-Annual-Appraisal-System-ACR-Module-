# Faculty Annual Appraisal System (ACR Module)

A faculty-side appraisal system developed to digitally collect and manage teacher academic records.

## Features
- Structured faculty data entry forms
- Backend processing using Java
- MySQL database integration
- CRUD functionality for record handling

## Tech Stack
- HTML
- CSS
- JavaScript
- Java
- MySQL

## Purpose
Developed as a college academic module to digitize faculty annual confidential report submission.
